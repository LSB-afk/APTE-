# 3. 자료 및 전처리 · 4. 분석 방법론 (초안)

## 3. 자료 및 전처리

### 3.1 자료 개요와 타겟 정의

한국도로공사 고속도로 교통사고 자료(2011~2019) 83,297건을 분석하였다`[확인 필요:
자료 출처·제공기관 공식 표기]`. 인명피해 발생은 사고별 사망·중상·경상 인원수를
이용해 다음과 같이 정의하였다.

> injury = 1 if (사망 + 중상 + 경상) ≥ 1, else 0

인명피해 사고는 8,993건(10.80%)으로, 대물피해 사고 74,304건(89.20%)과 약 8.3:1의
클래스 불균형을 이룬다. 인명피해 사고의 내부 구성은 최고 심각도 기준 사망사고
1,968건(2.36%), 중상사고 1,635건(1.96%), 경상사고 5,390건(6.47%)이다.

### 3.2 사고 전 변수와 누수 차단

입력변수는 사고 전(pre-accident) 시점에 관측 가능한 정보로 한정하였다. 사고유형
(차-사람 등), 사고차량수, 피해 규모, 사고처리 시간 등 사고 결과로부터 파생되는
변수는 타겟 정보를 누수시키므로 배제하였다. 자료 검증 과정에서 운전시간 변수는
전체의 99.9%가 0이고 나머지가 타임스탬프 오입력으로 확인되어 제거하였다.

18개 후보 변수에 대해 표기 정규화(중복 표기 통합), 결측 처리(결측률 5% 미만
최빈값 대치, 5% 이상 '미상' 범주 유지), 희소 범주 통합(0.5% 미만 → '기타')을
수행하였다.

### 3.3 공선성 진단과 변수 선별 (18개 → 14개)

변수 수준에서는 bias-corrected Cramér's V를 산출하여 V≥0.6인 쌍 중 타겟 연관성이
낮은 변수를 제거하였다(주야구분: 조명시설과 V=0.710 — '해당없음(주간)' 범주가 주간
정보를 내포). 이어 타겟 연관성 하위 변수(주말여부, 포장구분, 성별)를 제외하여 14개
변수를 확정하였다. 최빈범주를 기준으로 한 원-핫 인코딩 후 더미 수준 VIF는 최대
5.46(평균 1.48)으로 다중공선성 기준(VIF>10)을 초과하는 더미는 없었다.

최종 14개 변수: 주사고원인, 교통장애요인, 발생지점, 운전자상태, 사고시도로환경,
원인차차종, 날씨, 조명시설, 노면상태, 연령구분, 사고직전차량조작, 차종구분,
절성토구분, 평면선형 (표 1)

## 4. 분석 방법론

전체 절차는 ① 예측·탐색 단계(XGBoost-SHAP)와 ② 확인 단계(로지스틱·음이항 회귀)로
구성된다(그림: 방법 흐름도 `[확인 필요: 제작]`).

### 4.1 클래스 불균형 처리와 XGBoost 학습

다수 클래스(대물피해)를 소수 클래스 크기에 맞춰 무작위 추출하는 1:1 랜덤
언더샘플링(8,993:8,993)을 적용한 뒤, 층화 80/20으로 학습·평가 자료를 분할하였다.
학습 자료의 15%는 검증용으로 분리하여 조기종료(early stopping)에 사용하였다.
언더샘플링의 표본 의존성을 통제하기 위해 서로 다른 시드로 5회 반복하여 성능의
평균과 표준편차를 보고한다. XGBoost 초매개변수는 max_depth=6, learning_rate=0.05,
subsample=colsample_bytree=0.8, min_child_weight=5이며 격자 탐색으로 적정성을
확인하였다. 비교 기준으로 동일 분할의 로지스틱 회귀, 전체 불균형 자료 +
scale_pos_weight + 임계값 최적화 설계, SMOTEN 오버샘플링 설계를 함께 평가하였다.
모든 변수가 범주형이므로 보간 기반 SMOTE 대신 범주형 전용 SMOTEN을 사용하였다
(Chawla et al., 2002의 명목형 확장).

### 4.2 SHAP 기반 탐색 (1단계)

TreeExplainer로 SHAP 값을 산출하여 전역 중요도(mean |SHAP|)와 방향(조건 존재 시
평균 SHAP)을 구하고, SHAP interaction values로 상호작용 후보를 탐색하였다. 상호작용
강도는 두 가지로 산출한다: (A) 전체 표본 평균 |상호작용|(전체 기여, 빈도 가중),
(B) 두 조건 동시발생 표본에서의 평균 |상호작용|(건당 강도). 동일 변수에서 파생된
더미 쌍은 상호배타 구조의 인공물이므로 제외하였다.

### 4.3 확인적 통계 검정 (2단계)

**변수 수준**: 카이제곱 검정과 Cramér's V. **범주 수준**: 전체 83,297건에 대한
다변량 로지스틱 회귀로 오즈비(OR)·95% 신뢰구간·p값을 추정하였다(기준범주=최빈범주,
완전분리 더미는 추정 제외·각주 처리). **사상자 수 척도**: 사고당 사상자 수(0~26명)
에 대한 음이항 회귀로 발생률비(IRR)를 추정하였다(과산포 α=3.43, 모멘트 추정 고정
GLM-NB `[확인 필요: MLE 수렴 실패로 인한 대체 — 각주 서술 수위]`).

**상호작용 확인**: 탐색 단계의 후보 쌍(랭킹 A·B 상위 합집합)마다 주효과 14개
변수를 모두 통제한 로지스틱 모형에 상호작용항을 추가하고 Wald 검정을 수행한 뒤,
Benjamini-Hochberg 절차로 FDR을 보정하였다(동시발생 100건 미만 쌍은 검정 제외).

**견고성 검증**: SHAP 방향 ↔ 로지스틱 계수 부호, 로지스틱 ↔ 음이항 계수 부호의
일치율을 산출하여 세 모형의 결론이 수렴하는지 확인하였다.

### 4.4 평가 지표와 해석 범위

Accuracy, Precision, Recall, F1-score, ROC-AUC, PR-AUC를 보고한다. 불균형
자료에서 Accuracy는 다수 클래스 예측만으로도 높아지므로 해석에서 배제하며, 균형
평가 지표는 50:50 기저율 기준임을 명시한다(실제 유병률 10.8% 환경의 운영 지표는
보조 설계 참조).
